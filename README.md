# coding-challenge

## run local

### setup mongodb
  -  docker-compose up -d mongo

### install package
  - npm install

### setup env
  - cp .env.example .env

### start server
  - npm run dev

### Access to API document
  - To access the API documentation using this link http://localhost:3000/api-doc, you also change the port 3000 to any number

### RUN TESTS
  - npm run test
